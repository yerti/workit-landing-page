### What I learned

en este ejercicio aprendi a como utilizar el clip path en css:

```html
<div class="path-morado"></div>
```
```css
.path-morado {
    width: 100%;
    height: 100px;
    background-color: var(--dark-purple);
    clip-path: ellipse(50% 11% at 50% 50%);
    margin-top: -52px;
}
```
